# Business Sales Analytics Dashboard

## Objective
Analyze business sales performance using Power BI.

## Tools Used
- Power BI
- Excel

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Dashboard Features
- Monthly Sales Trend
- Regional Sales Performance
- Profit Contribution by Category
- Top 10 Products by Sales
- State-wise Sales Distribution
- Interactive Filters

## Key Insights
- West region generated the highest sales revenue.
- Technology category contributed the highest profit.
- Sales peaked during year-end months.
- Top-performing products significantly increased revenue.
